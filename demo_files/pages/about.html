About This Project

This project started because of the developer's favorite mobile game, Brawl Stars.

The backend for this app is built using node.js and it utilizes express.js to create a server that can both fetch and serve files. The frontend is built ontop of the bootstrap framework in order to make a responsive and smooth user experience.

A typical request to the Brawl Stars API starts off after a user submits their player tag. Javascript will then execute and communicate with the server which will make a request to Brawl Stars API to fetch a JSON file. This will be sent back to the client side and then the web page will dynamically update with data and chart.js graphs displaying the data requested.
